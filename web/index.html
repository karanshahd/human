HUMAN : Hadoop Used Medical ANalytics

Basic Overview

Welcome to Project Human

Choosing to go to a doctor requires producing old records for better treatment of the patients. Our website is designed to allow patients to create a user profile for themselves where they can store their data securely over the cloud service. After creating the profile the users can view statistical and analytical data about current diseases and trends. We collect data about patients and doctors from various hospitals and clinics to store it in an secure and reliable manner. The system is designed to handle large amounts of data in a quick and reliable manner.

ELECTRONIC HEALTHCARE SYSTEM

What do we do?

Store

We collect information from all users and store it in our Hadoop database through inputs from medical professionals.

Analyze

The collected information is analysed using hive. Analytics takes place through data aggregation,normalisation and discretization functions.

Display

Each user can view his/her data collected by us. Also, he/she can view the analytical results processed by the system.
